# AuMiau Shop

Loja online simples para produtos de cães e gatos, criada com Next.js.

## Funcionalidades

- Home com produtos em destaque.
- Catálogo com busca, filtro por categoria, filtro por pet e ordenação por preço.
- Página individual de produto.
- Carrinho persistido no `localStorage`.
- Checkout de teste com resumo do pedido e número de pedido salvo no navegador.

## Rodando localmente

```bash
npm install
npm run dev
```

Depois acesse [http://localhost:3000](http://localhost:3000).

## Scripts

```bash
npm run dev
npm run lint
npm run build
npm run start
```

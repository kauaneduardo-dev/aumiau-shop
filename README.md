# AuMiau Shop

Loja online simples para produtos de cães e gatos, criada com Next.js.

## Funcionalidades

- Home com produtos em destaque.
- Catálogo com busca, filtro por categoria, filtro por pet e ordenação por preço.
- Página individual de produto.
- Carrinho persistido no `localStorage`.
- Checkout pelo WhatsApp com resumo do pedido.

## Usar na escola

Se você vai abrir o projeto em outro notebook, siga o passo a passo em [GUIA_ESCOLA.md](./GUIA_ESCOLA.md).

## Rodando localmente

```bash
npm install
npm run dev
```

Depois acesse [http://localhost:3000](http://localhost:3000).

## Scripts

```bash
npm run dev
npm run lint
npm run check
npm run build
npm run start
```
